1. Pattern/pattern
Repeating elements (color, shape, line)

https://ik.imagekit.io/71jz3gcgav/image_HJnPVEIzn.png

1. Accent
Highlight an important object by color, shape, or size. (For example, all the elements
are square, but one of them is round, the eye will automatically **guiding** there

https://ik.imagekit.io/71jz3gcgav/image_EvCeFTERB.png
1. Contrast
Compare different elements and create contrast between them. For example: Black/white,
clear/blurred

https://ik.imagekit.io/71jz3gcgav/image_0JF-G13BT.png

1. Balance
Each object has its own weight. Elements are balanced within a given framework. Our
goal is to balance the sides

https://ik.imagekit.io/71jz3gcgav/image_HlBu8U9Br.png

1. Proportion/Scale
Compare elements by size, thus emphasizing the more significant one

https://ik.imagekit.io/71jz3gcgav/image_SOXweIDvx.png

1. Rhythm/Dynamics
Elements can be arranged in such a way that the eye is led to a compositional
center, or important information in the picture

https://ik.imagekit.io/71jz3gcgav/image_90yDimLd2.png

All principles can be combined, but the more objects in the picture, the more complicated it becomes.
When you add a new object, make sure it fits into the overall composition. Therefore, it is very important to see the whole picture.

If the object is in 3D, then there are other sides to consider, making the task more complicated. And don't forget that it's okay to break the rules sometimes, but you have to do it with purpose.
